### 编译
```shell
#从官网下载合适的版本并解压
zlib_version=1.2.11
wget -O- "https://www.zlib.net/zlib-$zlib_version.tar.gz" | tar xz

#进入编译目录
cd zlib-$zlib_version

#设置 prefix
PREFIX=~/x64

#编译
./configure --prefix=$PREFIX
make install -j $(grep "cpu cores" /proc/cpuinfo | wc -l)
```

编译完之后，就可以删除源文件了
```shell
cd ..
rm -rf zlib-$zlib_version
```
