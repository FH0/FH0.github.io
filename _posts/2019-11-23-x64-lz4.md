### 编译
```shell
#从官网下载合适的版本
lz4_version=1.9.2
wget https://github.com/lz4/lz4/archive/v$lz4_version.tar.gz

#解压缩
tar xf v$lz4_version.tar.gz

#进入编译目录
cd lz4-$lz4_version

#设置库文件安装目录
LIB_dir=~/x64

#编译
make lib -j $(grep "cpu cores" /proc/cpuinfo | wc -l)
make install PREFIX=$LIB_dir
```

编译完之后，就可以删除源文件了
```shell
cd ..
rm -rf "lz4-$lz4_version" "v$lz4_version.tar.gz"
```
